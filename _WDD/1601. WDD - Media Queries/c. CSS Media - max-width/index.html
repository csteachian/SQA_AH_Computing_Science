Hello Advanced Highers!

Resize this browser tab to see the impact of this media query:

body { background-color: green; }

@media screen and (max-width: 992px) {

   body { background-color: orange; }

}

@media screen and (max-width: 600px) {

   body { background-color: red; }

}

As the max-width of the @media query is exceeded, another takes over until the width of the browser tab is > 992 px, then the body background colour is used.
